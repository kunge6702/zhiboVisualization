# Web 应用 + 订阅制，支持离线使用

直播信号路由规划工具采用 Web 应用形态，以订阅制作为商业模式，同时支持离线使用（PWA）。

**选择的理由：**

订阅制提供可持续的经常性收入，避免买断制"必须不断出大版本才能产生收入"的困境。Web 应用将获客摩擦降到最低——用户点击链接即可试用，无需下载安装，利于在直播行业圈内口碑传播。离线支持通过 PWA 实现，解决活动现场网络不可控的痛点。

**否决的方案：**

桌面应用 + 买断制。分发路径重（下载/安装/兼容性），收入不可预测，且未来扩展团队协作功能的技术难度极高。

**为未来留出的空间：**

Web 架构天然支持多用户协作，可从"个人工具"升级为"团队工具"，作为提价和提升客单价的杠杆。
